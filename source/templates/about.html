Networker is a platform designed to help users better network with each other in the job marketplace by allowing them different functionalities that enable better networking with others seeking jobs or hiring for jobs, and more effective management of job applications.


Some of the functionalities include: management of one's own profile and overview of others' profiles, overview of others' posts and evalaution of and commenting on them, creation of one's own posts, listing of job posting created by others with full information about the jobs' details, ability to apply for any one of the job postings, ability to track one's own applications with real-time status on them, ability to connect with other users and text message with them.


This platform is a web application developed to satisfy an Independent Study class at a university by a senior student. Under the hood, the web application is developed in Flask framework, which is a micro web framework written in Python programming language. This framework helps in rendering of HTML web pages styled by CSS stylesheets - no JavaScript, a web scripting language that runs on web browsers, is implemented in the development of the application - which explains why pages in the app reload at each button click. The backend is developed in Python programming language too, with the database being implemented at the Firebase Real Time Database server.


As a web application, this platform is at its initial mock-up stage and is under further improvement. As a mock-up version, the app might contain bugs or other programming inferiorities.
